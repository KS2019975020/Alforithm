문제 인식:
N개의 수가 주어지면 이 수들을 오름차순으로 정렬

문제 유형 파악:
배열에 수들을 저장하고 오름차순으로 정렬한 뒤에 출력

알고리즘 나열:
1. Scanner 사용, 변수 n에 값을 받음
2. 배열 num을 만들고 for문 사용, num 배열에 값을 받음
3. key에 배열의 인덱스 i의 값을 넣음
4. while문 사용, j >= 0 && num[j] > key일때 반복
5. 배열의 j + 1 인덱스에 j 인덱스 값을 넣음
6. j는 -1
7. 배열의 j + 1의 인덱스에 key 값을 넣음
8. for문 이용, 배열의 값을 순서대로 출력
