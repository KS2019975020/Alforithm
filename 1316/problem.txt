문제 인식:
그룹 단어의 갯수를 출력해야 하고 단어가 떨어져 있으면 그룹 단어가 아니다

문제 유형 파악:
단어의 갯수를 받고 그룹 단어인지 확인하는 method를 만들어 확인하고 카운트

알고리즘 나열:
1. 단어의 갯수 N을 입력 받음
2. 변수 cnt를 선언
3. for문 사용, i = 0, i < N일때 반복
4. 변수 word에 단어 입력 받음
5. 그룹 단어 인지 판별하기 위해 groupWord method를 만듬
6. groupWord에서 boolean 타입의 배열 checked를 만듦
7. 배열의 크기는 알파벳 소문자 갯수와 동일하게 26개로 함
8. int prev = -1 선언
9. for문 사용, true인지 false인지 구분
10. idx 변수 선언, idx = ch - 'a';
11. if문 사용, 연속하지 않은 문자이고 이미 등장한 문자가 떨어져서 나타나면 false를 반환
12. checked 배열에 사용 여부 저장
13. 이전 문자의 인덱스를 업데이트
14. 그룹 단어를 판별했을 경우 count 증가
15. 그룹 단어 갯수 출력
