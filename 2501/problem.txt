문제 인식:
변수 N, K에 사용자가 값을 넣고 N의 약수들 중 K번째로 작은 약수를 출력

문제 유형 파악:
배열을 만들고 for문을 이용해 배열에 값을 넣고 K번째로 작은 약수를 출력

알고리즘 나열:
1. 변수 N, K에 Scanner이용해 정수를 입력
2. ArrayList 배열을 만듦
3. for문 사용
4. 변수 i를 만들고 N이 i보다 작거나 같을 때 반복
5. i는 for문이 반복할 때마다 1씩 증가
6. N % i == 0이면 배열에 i를 추가
7. K가 배열의 크기보다 작거나 같으면 K번째 작은 약수를 출력
8. 그 외에는 -1을 출력
