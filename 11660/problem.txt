문제 인식:
N * N개의 수가 N * N 크기의 표에 존재하고, 그들의 좌표 (x1, y1)에서 (x2, y2)까지 합을 구한다.

문제 유형 파악:
이차원 배열을 만들어 수를 입력받고 합을 구한 다음 출력

알고리즘 나열:
1. 표의 크기 N과 합을 구해야 하는 횟수 M에 입력 받음
2. 이차원 배열 array를 생성
3. 이중 for문 사용, 행은 i, 열은 j이고 배열에 수를 입력 받음
4. 이차원 배열 a_sum을 생성
5. 이중 for문 사용, 2차원 누적합을 a_sum 배열에 넣음
6. for문 사용, x1, y1, x2, y2를 받음
7. 변수 sum에 위에 입력한 좌표 사이에 포함된 수들의 합을 넣음
8. sum을 출력
9. 이는 i < M일때 실행하며 i = 0부터 시작해서 반복 1회 끝나면 i++
