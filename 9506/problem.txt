문제 인식:
변수 n이 완전수인지 아닌지 판단

문제 유형 파악:
약수들을 반환해주는 함수를 이용하여 나머지를 계산하고 결과 출력
완전수가 아닌경우에는 "is NOT perfect"를 출력하게 함

알고리즘 나열:
1. 약수들을 반환하는 메소드 function을 만듦
2. n을 2로 나눈 값까지만 반복하게 함
3. if문으로 약수를 찾으면 리스트에 넣음
4. while문 조건을 true로 함
5. main에서 Scanner로 변수 n에 정수를 받음
6. 만약 n == -1이면 break로 반복문을 벗어남
7. list에 function 메소드를 이용하여 약수들을 받음
8. 변수 sum을 추가
9. 약수들의 합을 계산
10. sum == n일 경우 원하는 형태의 값을 출력하는 과정을 거침
11. 그 외에 경우는 "is NOT perfect가 출력되도록 함
